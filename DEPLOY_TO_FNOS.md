# 部署 "雪沐江南" 图库到飞牛OS (FnOS) 指南

这是一个利用飞牛OS的Docker功能和IPv6公网IP，零成本将网站发布到互联网的操作手册。

## 第一步：上传文件到飞牛OS

1.  **打开飞牛OS的文件管理器**。
2.  在你的 Docker 常用数据目录下创建一个新文件夹，例如命名为 `web_gallery`。
3.  将项目文件夹里的**所有内容**上传到飞牛OS的这个新文件夹中。
    *   确保结构如下：
        ```text
        /web_gallery/
        ├── index.html          (入口文件)
        ├── web_data.json       (数据文件)
        ├── assets/             (图片文件夹)
        └── assets_extracted/   (图片文件夹)
        ```

## 第二步：在飞牛OS上部署 Nginx 容器

1.  打开飞牛OS的 **Docker（应用中心/容器管理）**。
2.  在镜像仓库搜索 `nginx`，拉取 `latest` 官方镜像。
3.  **创建/运行容器**，配置如下：
    *   **容器名称**：`my-web-gallery` (或者你喜欢的名字)
    *   **硬件渲染**：不需要。
    *   **启动命令 (Command)**：`/bin/sh -c "sed -i 's/user  nginx;/user  root;/g' /etc/nginx/nginx.conf && nginx -g 'daemon off;'"` (核心：修改配置以root运行)
    *   **网络模式**：
        *   **推荐方案 (Bridge模式)**：
            *   将容器端口 `80` 映射到本地端口 `8888` (或任意空闲端口)。
    *   **存储空间（目录映射）**：
        *   **主机路径**：选择你刚才上传文件的文件夹 `/你的路径/web_gallery`
        *   **容器路径**：`/usr/share/nginx/html` (必须完全一致)
        *   权限：`只读` (推荐)

4.  启动容器。

## 第三步：测试内网访问

打开浏览器，输入 `http://<飞牛OS的内网IP>:8888`。
如果能看到“雪沐江南”的页面，说明部署成功。

## 第四步：设置 IPv6 公网访问

1.  **获取 IPv6 地址**：
    *   在飞牛OS的网络设置里查看它的 **IPv6 全局单播地址**（通常以 `240e`, `2409`, `2408` 开头）。
2.  **路由器防火墙设置**（关键）：
    *   登录路由器后台。
    *   找到 **防火墙** 或 **IPv6 防火墙**。
    *   **添加规则**：
        *   协议：TCP
        *   源地址：任意（留空或 `::/0`）
        *   目标地址：飞牛OS的 IPv6 地址
        *   端口：`8888`
        *   动作：允许 (Allow)
3.  **互联网测试**：
    *   断开手机 WiFi，使用 4G/5G 流量。
    *   在手机浏览器输入：`http://[你的飞牛OS IPv6地址]:8888`
    *   *注意：IPv6地址在浏览器里必须用中括号 `[]` 包起来。*

## 进阶建议 (DDNS)

*   建议在飞牛OS上配置 **DDNS-Go**。
*   配合域名服务商（阿里云/腾讯云/Cloudflare），将域名解析到你飞牛OS的最新 IPv6 地址。
*   这样即可通过域名访问，无需记忆IP。
